## Document Summary

The Problem Statement defines the *why* behind the project. It covers:

- **The problem being solved** — Most people lack intuition for non-decimal numeral systems. Existing educational tools are passive (textbooks, videos) and fail to build genuine fluency through practice.
- **Proposed solution** — A competitive, real-time multiplayer card game (*Crazy 1-0's!*) that forces players to perform arithmetic in alternative bases as a core part of gameplay, not as an optional exercise.
- **Project goals** — Deliver a fully playable 1v1 web game supporting at minimum Dozenal (Base-12) and Decimal (Base-10), with correct scoring, real-time synchronization, and an arithmetic challenge mini-game.
- **Stretch goals** — Support for additional bases (Octal, Hexadecimal), multi-round match tracking, leaderboards, and richer player profiles.
- **Stakeholders** — Students, educators interested in numeral system literacy, and players who enjoy casual competitive card games.
